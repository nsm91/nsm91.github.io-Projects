NEIU Masters Projects

Below is a showcase of just some of the projects I have completed in regards to GIS while attending NEIU for both a Masters and Certificate of GIS proficiency.

ArcMap ModelBuilder

Purpose

In this exercise we created a model to facilitate a mailing list based on resident proximity to user selected streets. The size of the buffer, where the mailing list would be saved to, and what it would be named were all user defined parameters.

Arcpy Python Scripting

Purpose

Utilizing "for" loops to facilitate batch processing, we created a script to handle multiple operations to multiple feature classes all at once. This exponentially quicker than doing it manually, and can be replicated across multiple users.

Rule Based Tree Layer

Goals

Since the actual tree data was the focus and driver of this entire project, it is only fitting that they comprise the most robust portion of the map. As such I utilized three levels of rule based options (for styling, rendering, and labeling) to deliver the information in the most immediately useful fashion possible. Only currently standing trees would appear at full zoom, with the conventional symbol, and with a label. All historic trees would only render at a zoomed in scale, as smaller and less pronounced red dots, and would do so without labels, unless user selected, so as to not crowd the map.

Highlights

  • Rule Based Styles
  • Rule Based Renderings
  • Rule Based Labels
